发版说明

V4.7.0

【功能更新】
  • 调整 ISerialPort.Write() 的单次最大数据写入量为512字节。
  • 支持RGB12格式图像的渲染和保存。
    • 支持渲染RGB12像素格式的图像。
    • 支持调用IImageSaver.SaveImageToFile() [1/2]、IImageSaver.SaveImageToFile() [2/2]保存RGB12格式的图像。
  • 支持快速FileAccess功能。若相机和其固件支持，则可通过该功能，大幅提高文件的导入导出速率。
【功能更新】
  • IImageRender.DisplayOneFrame() 支持渲染 PixelType_Gvsp_YUV420SP_NV12 和 PixelType_Gvsp_YUV420SP_NV21 格式。
  • IPixelTypeConverter.ConvertPixelType() 支持将部分Bayer8（包含BayerRG8、BayerGR8、BayerGB8、BayerBG8）像素格式图像转换为RGB8 Planar像素格式图像。
  • 事件相关功能优化。
    • 单进程下，支持同一采集卡的多个实例同时注册事件。在同一进程下，假设您分别对其实例A和实例B调用 IEventGrabber.SubscribeEvent() 注册同一事件M，实例A和实例B会同时接收到事件M。
      注解
      对同一实例多次注册同一事件时，旧注册的回调函数和自定义信息会被覆盖。
    • SDK支持接收CoaXPress相机事件。
  • StreamExceptionType 下新增0x4008（ImageBufferOverflow）表示设备发送的图像数据大小超过了图像缓存区容量。
  • VisionPro插件支持VP9.23和VP9.24。
  • 优化采集卡事件性能，降低CPU占用。
