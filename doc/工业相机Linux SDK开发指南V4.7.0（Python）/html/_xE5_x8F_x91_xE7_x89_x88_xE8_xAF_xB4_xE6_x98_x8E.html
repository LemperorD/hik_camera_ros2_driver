发版说明

V4.7.0

【功能新增】
  • 支持RGB12格式图像的渲染和保存。
    • 支持通过GDI、D3D和OpenGL模式渲染RGB12像素格式的图像。
    • 支持调用 MV_CC_SaveImageEx3() ， MV_CC_SaveImageToFileEx() 和 MV_CC_SaveImageToFileEx2() 保存RGB12格式的图像。
  • MV_CC_DisplayOneFrameEx() 和 MV_CC_DisplayOneFrameEx2() 支持渲染 PixelType_Gvsp_YUV420SP_NV12 和 PixelType_Gvsp_YUV420SP_NV21 格式。
    注解
  • MV_CC_ConvertPixelTypeEx() 支持将Bayer像素格式图像转换为RGB Planar像素格式图像。
  • 事件相关功能优化。
    • 单进程下，支持同一采集卡的多个句柄同时注册事件。在同一进程下，假设您分别对其句柄A和句柄B调用 MV_CC_RegisterEventCallBackEx() 注册同一事件M，句柄A和句柄B会同时接收到事件M。
      注解
      对同一句柄多次注册同一事件时，旧注册的回调函数和自定义信息会被覆盖。
    • SDK支持接收CoaXPress相机事件。
【示例程序更新】
新增如下示例程序：
  • Recording.py ：演示如何将相机拍摄到的画面存储为视频录像。
  • Grab_LineStart.py ：演示在指定线程设置线阵相机取流。
  • BasicDemoLineScanCamera.py ：演示线阵相机的基础使用方式，包括初始化SDK、枚举设备、创建实例和句柄等。
