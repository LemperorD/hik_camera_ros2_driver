◆ MV_FG_DisplayOneFrame()

MV_FGCTRL_API int __stdcall MV_FG_DisplayOneFrame ( IN IMAGEHANDLE  hImage,
IN void *  hWnd,
IN MV_FG_DISPLAY_FRAME_INFO *  pstDisplayFrameInfo 
)

显示一帧图像

参数
hImage[IN] 图像句柄(可以使用采集卡/相机/流句柄)
hWnd[IN] 窗口句柄
pstDisplayFrameInfo[IN] 显示的图像信息
返回
成功，返回MV_FG_SUCCESS；失败，返回状态码 。
备注
在Linux平台进行远程显示时，需要OpenGL支持远程直接渲染模式，OpenGL版本高于4.0版本，并且显卡驱动支持高版本OpenGL。

◆ MV_FG_DisplayOneFrameEx()

MV_FGCTRL_API int __stdcall MV_FG_DisplayOneFrameEx ( IN IMAGEHANDLE  hImage,
IN void *  hWnd,
IN MV_FG_DISPLAY_FRAME_INFO_EX *  pstDisplayFrameInfo,
unsigned int  enRenderMode 
)

显示一帧图像

参数
hImage[IN] 图像句柄(可以使用采集卡/相机/流句柄)
hWnd[IN] 窗口句柄
pstDisplayFrameInfo[IN] 显示的图像信息
enRenderMode[IN] 渲染方式，0-OpenGL
返回
成功，返回MV_FG_SUCCESS；失败，返回状态码 。
备注
在Linux平台进行远程显示时，需要OpenGL支持远程直接渲染模式，OpenGL版本高于4.0版本，并且显卡驱动支持高版本OpenGL。 只有OpenGL模式支持渲染超过4G大小的图像(需要显卡支持)
